Installation
============

Requirements
------------

- ROS Noetic
- Python 3
- Additional dependencies: OpenCV, NumPy

Installation Steps
------------------

Clone and build the SimpleBot repository::

    cd /<path to catkin_ws>/src
    git clone https://github.com/example/simplebot.git
    cd /<path to catkin_ws>/
    catkin build
    source devel/setup.bash
